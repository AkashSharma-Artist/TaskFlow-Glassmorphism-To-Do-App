# TaskFlow – Glassmorphism To-Do App

![](https://github.com/AkashSharma-Artist/TaskFlow-Glassmorphism-To-Do-App/blob/main/ChatGPT%20Image%20Jul%2010%2C%202026%2C%2012_52_24%20AM.png)

TaskFlow is a fully functional, single-page To-Do application built with 
HTML5, CSS3, and vanilla JavaScript. It features a modern glassmorphism 
UI design with real-time progress tracking, persistent storage, and 
a fun confetti celebration when all tasks are completed.

## Features
- Add, edit, and delete tasks
- Custom animated checkbox with strike-through on completion
- Real-time progress bar + task counter (e.g. "3 / 5")
- 🎉 Confetti celebration animation when all tasks are completed
- Data persistence using **localStorage** (tasks stay saved on refresh)
- Empty-state illustration when no tasks exist
- Glassmorphism design — blurred glass card, backdrop blur, soft shadows
- Fully responsive (mobile-friendly)

## Tech Stack
- HTML5
- CSS3 (Glassmorphism, backdrop-filter, media queries)
- Vanilla JavaScript (DOM manipulation, event handling, localStorage)
- Font Awesome (icons)
- tsParticles Confetti (celebration effect)
- Google Fonts (Jost)
